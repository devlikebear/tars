# tars-ui

React/TypeScript(INK) 기반 TARS TUI 클라이언트.

## Quick start

```bash
cd tars-ui
npm install
npm run dev -- --server-url http://127.0.0.1:8080 --verbose
```

## Options

- `--server-url` : `tarsd` API URL (기본값: `http://127.0.0.1:8080`)
- `--session` : 시작 세션 ID
- `--verbose` : 우측 패널에 디버그 이벤트 표시
